Book Lending and Returning Machine

Duration: 3 Months
Type: Minor Project
Technology Used: Python and PHP
Description: This machine helps a user to self issue and self deposit the books without the help of librarian in a library. This machine also has a separate software which is compatible to run with RFID Technology and provides best security system to the library.

Solar- Electric Vehicle

Duration: 6 Months
Type: Major Project
Technology Used: Solar and Electrical
Description: This vehicle is made to run without using any kind of fossil fuels. This vehicle runs by using electricity and the batteries are rechargeable using solar power taken from the solar panels placed on top of vehicle.

Feedback on Project


Egg Catcher Game

Duration: 4 Weeks
Type: Minor Project
Technology Used: Python
Description: This game has a bowl and many eggs which are generated randomly. If a person is able to catch the eggs in the bowl using the arrow keys, the score will be increased. If three eggs miss the bowl and fall down, then the game ends.
